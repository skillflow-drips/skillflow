# Security Policy

SkillFlow touches wallet authentication, escrow state, and off-chain payment records. Please report security issues privately instead of opening a public issue.

## Scope

- SEP-10 wallet authentication
- SEP-7 transaction generation
- Soroban escrow contract logic
- Prisma-backed token balances and transaction records
- IPFS upload handling

## Reporting

Send a private report to the maintainers with:

- Impact summary
- Reproduction steps
- Affected commit or version
- Suggested mitigation, if known

Maintainers should acknowledge reports within 72 hours and avoid public disclosure until a fix is available.
